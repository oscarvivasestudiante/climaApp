## 🖥️ Instalacion

1. Clone o repositório:
   ```
   git clone https://github.com/lipesshw/weatherapp
   cd weatherapp/
   ```

2. Instale las dependencias:
   ```
   npm install
   ```

3. Finalmente ejecute el proyecto:
   ```
   npm run dev
   ```
